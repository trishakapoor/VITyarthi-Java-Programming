# VITyarthi-Java-Programming
The Assignment Deadline Manager is a console-based Java application that helps students organize and track their assignments efficiently. It allows users to add, view, and manage tasks with deadlines and priorities, helping improve time management and reduce missed submissions.

# Assignment Deadline Manager
A CLI-based Java application to manage assignments and deadlines.

## Features
- Add assignments
- View all assignments
- Track upcoming deadlines
- Identify overdue tasks
- Delete assignments
- File-based storage

## Tech Stack
- Java SE (JDK 8+)
- Collections Framework
- File Handling
- OOP Concepts

## How to Run

### Compile
javac -d . src/Main.java src/models/*.java src/services/*.java src/utils/*.java

### Run
java Main
